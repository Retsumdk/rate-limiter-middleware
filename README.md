# rate-limiter-middleware

Token bucket rate limiting for Node.js APIs

## Features

- Production-ready code
- TypeScript/Python with full type safety
- Comprehensive error handling
- Built with Zo Computer by The BookMaster

## Installation

```bash
git clone https://github.com/Retsumdk/rate-limiter-middleware.git
cd rate-limiter-middleware
bun install
```

## Usage

```bash
bun run src/index.ts --help
```

## Configuration

Create `config.json` (TypeScript) or `config.yaml` (Python) for custom settings.
